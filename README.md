# Test Scenario    

Send a PUT request to the URL https://jsonplaceholder.typicode.com/posts/70 with the following body.

Test whether the returned response has the same body as the one provided below.    

Request Body:    

      {
        "title":"John",
        "body":"Hi",
        "userId":10,
        "id":70
      }    

Expected Data:

      {
        "title":"John",
        "body":"Hi",
        "userId":10,
        "id":70
      }
